# Netflix-Data-Analysis-using-SQL
![Netflix logo](https://github.com/aireenfatma/Netflix-Data-Analysis-using-SQL/blob/main/netflix-logo.png)

#Project Objective
The primary objective of this project is to analyze the Netflix dataset using PostgreSQL to uncover strategic insights regarding content trends, regional distribution, and artistic talent profile metrics.

Specifically,the project aims to:

- Quantify Catalog Balance: Evaluate the distribution metrics between available Movies and TV Shows.
- Track Maturity & Ratings: Pinpoint the most frequent target demographics across content categories.
- Examine Global footprints: Clean up multi-valued regional strings to rank the top content-producing nations.
- Identify Production Velocity: Measure how quickly library assets are being added over rolling year ranges, specifically focusing on regional targets like India.
- Analyze Content Characteristics: Segment titles based on target runtimes, long-form multi-season statuses, specific creative personnel, and textual description keyword patterns.
